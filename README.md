# Apex Legends Stat Tracker

> Full stack Vue/Express app using the Tracker Network API to track player stats

This is the project from the [YouTube](https://www.youtube.com/watch?v=8z2qRln9tnc).

## Quick Start

Register and get your API key from
https://tracker.gg/developers

Add the API Key to _config.env_

```bash
# Install Express dependencies
npm install

# Install Vue dependencies
cd client
npm install

# Serve
npm run dev

# Build for production
cd client
npm run build
# Gets put in /public
```
